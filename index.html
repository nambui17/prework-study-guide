    CSS

  • To add a css style to the html, use function "touch 'filename'"
  • Remember to checkout a new branch and name it something before changing main
  • Open VS code after using touch function to find the css
  • For css the padding is the area around the element inside and margin is area around the entire thing can specify image size, font size, etc.
  • Can also change the color of the background and header color
  • Link the css code to the html file by adding it in the metadata section
  • This is the link function with "rel" function specifying the relationship between the current document and the linked document
  • The "href" function specifies what the name of the css file is. Start with "./assets/'filename'"
  • The period in front of "card" in ".card" designates it as a class selector. Any HTML element assigned to the class "card" will gain these properties from the CSS
  • "Width" "margin" "border" are self explanatory
  • The box-shadow CSS property adds shadow effects around the element's frame
  • Can assign color with "blue" or "gray" but can also assign colors with hexadecimal number (hex #)
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

    Git

  • git status checks what branch we're currently on
  • To go to a specific branch use git checkout "Branch name here"
  • To add changes to a feature to the main branch use the command "git add -A" The add -A means add all changes
  • Once all changes have been staged we can then commit them to the repository by using command "git commit -m "Insert note about change e.g. added code to HTML file"

    • JavaScript

  • To add a JavaScript file to the html, it is the same as css use the "touch" function
  • Set variables for using the function "var 'name of topics here' = 'name whatever'"
  • Use the function "console.log("Input message here");" to log into the console and can be seen with google chrome inspect element feature
  • if and else if statements similar to MATLAB
  • Start with "if (topic=== 'blank') {}else if (topic=== 'blank2') {}" and so on and end with a close bracket on the next line
  • For for loops, similar to MATLAB for loops except can add how much iterates in the function
  • For loop examples "for(var x=0; x< shapes.length; x++) { // code block }
  • The var x=0 tells the starting point for x which is defined in a for loop
  • The shapes is the name of the array which must be previously defined
  • The .length after the name of the array is means the value is length or size of the array similar to length(x) in MATLAB
  • x++ is shorthand for x+1
  • The array is defined through the beginning "var" function that sets the array
  • Arrays are defined as in square brackets and apostrophes e.g. ['HTML','CSS','Git',JavaScript']
  • Add JavaScript functions using the keyword "function" similar to how var was used.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
